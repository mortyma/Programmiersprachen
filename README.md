Programmiersprachen
===================

Programmiersprachen VU, TU Wien SS2014

All assignments contain a makefile for easily building and running them.


## Assignment 2
### Requirements
 - Ruby 2.0
 - Rubygems: rubygems.org
 - Parslet: kschiess.github.io/parslet

### Test
``` ./test.rb ```

### Demo
``` ./sorter.coord ```
